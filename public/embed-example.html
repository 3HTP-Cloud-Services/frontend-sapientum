Sapientum Embed Example

Basic Embed

Here's how to embed Sapientum into your website:

<!-- 1. Add the embed script to your page -->
<script src="https://your-server.com/embed.js"></script>

<!-- 2. Create a container element -->
<div id="sapientum-container"></div>

<!-- 3. Initialize the embed -->
<script>
  const embed = SapientumEmbed.create('#sapientum-container', {
    width: '100%',
    height: '600px',
    server: 'https://your-server.com',
    theme: 'light'
  });
</script>

Embed will load here when you click "Load Embed"
