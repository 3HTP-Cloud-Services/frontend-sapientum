Sapientum AI Embed

Examples and resources for embedding Sapientum AI in your website

Quick Start

Add the following code to your website to embed Sapientum AI:

<\!-- 1. Add the embed script to your page -->
<script src="https://sapientum-test.3htp.cloud/embed.js"></script>

<\!-- 2. Create a container element -->
<div id="sapientum-container"></div>

<\!-- 3. Initialize the embed -->
<script>
  const embed = SapientumEmbed.create('#sapientum-container', {
    width: '100%',
    height: '600px',
    server: 'https://sapientum-test.3htp.cloud',  // Change to your Sapientum server
    theme: 'light'
  });
</script>
